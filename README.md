# fruit-clustering
Use various clustering methods to cluster fruit types.

現代社會中，健康飲食及營養狀況逐漸引起人們的重視，諸如「無麩質飲食」、「生酮 飲食」、「地中海飲食」等飲食計畫開始進入大眾的視野，然而水果往往是各大計畫中不被排斥的種類，與其盲目攝取多樣水果來補充營養素，不如根據缺乏的營養對症下藥。因此本研究想利用營養成分對水果分群，提供大眾作為飲食指南參考。
